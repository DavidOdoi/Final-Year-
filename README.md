
  # Freight Web App Landing Page

  This is a code bundle for Freight Web App Landing Page. The original project is available at https://www.figma.com/design/f027tp02RMS4IVjVD598QY/Freight-Web-App-Landing-Page.
  

  ## Running the code
  Clone the repo
  Open it in a code editor(  Vsscode preferable
  

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  Click or copy the link and paste it in your prefered browser.
